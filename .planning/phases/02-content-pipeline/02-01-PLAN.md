    1. Export `NoteData` interface:
       ```typescript
       export interface NoteData {
         title: string
         tags: string[]           // without # per MKDN-02
         source?: string          // "Переслано из канала X"
         originalUrl?: string     // video URL
         videoYaDiskUrl?: string  // yadisk video link
         date: string             // YYYY-MM-DD
         type: string             // текст | фото | видео | пересланное
         contentHash: string      // sha256
         text?: string            // note body text
         attachments: string[]    // ["attachments/img_xxx.jpg"] per PHOT-03
       }
       ```

    2. Export `buildMarkdown(data: NoteData): string`:
       - Open frontmatter with `---`
       - If tags.length > 0: `tags:\n  - tag1\n  - tag2` (YAML list without #, per MKDN-02/TAGS-04)
       - If source: `source: "${escapeYaml(source)}"` (always quoted — may contain colons)
       - If originalUrl: `video: "${escapeYaml(originalUrl)}"`
       - If videoYaDiskUrl: `video_yadisk: "${escapeYaml(videoYaDiskUrl)}"`
       - Always: `date: ${date}`, `type: ${type}`, `content_hash: "${contentHash}"`
       - Close frontmatter with `---`
       - Blank line
       - `# ${title}` (H1 heading — title lives here, NOT in frontmatter to avoid YAML colon issues)
       - Blank line
       - If text: the text content, then blank line
       - For each attachment: `![](${att})` (standard markdown, NOT wiki-link per anti-patterns in research)
       - Join with `\n`

    3. Internal `escapeYaml(s: string): string`:
       - Replace `\` with `\\`, then `"` with `\"`
       - This is minimal — only needed for double-quoted YAML strings

    Import nothing from config.ts or webdav.ts — this is a pure function module with zero side effects.
