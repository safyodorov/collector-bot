    - Construct full path: `${VAULT_PATH}${folderPath}${filename}`
    - Call `exists(fullPath)` from webdav.ts
    - Return boolean
    - Wrap in try/catch — on error, log warning and return false (don't block save on PROPFIND failure)

    **3. Export `saveEntry(data, photoBuffers): Promise<SaveResult>`** (per PHOT-02, PHOT-03, MKDN-01)
    ```typescript
    export async function saveEntry(
      data: {
        title: string
        text: string
        contentType: ContentType
        source: string
        originalUrl: string
        videoYaDiskUrl: string
        hash: string
        folderPath: string       // e.g. "/Бизнес/WB/"
        tags: string[]
      },
      photoBuffers: { name: string; buffer: Buffer }[]
    ): Promise<SaveResult>
    ```

    Implementation:
    a. Compute date: `new Date().toISOString().slice(0, 10)`
    b. Build filename: `buildFilename(data.title, date)`
    c. Build note path: `${VAULT_PATH}${data.folderPath}${filename}`

    e. Build NoteData object with all fields
    f. Generate markdown via `buildMarkdown(noteData)`
    g. Upload markdown: `await putFile(notePath, markdown)`
    h. Return `{ path: data.folderPath + filename, title: data.title }`

    Photos uploaded BEFORE markdown so the attachment exists when Obsidian tries to render the note.

    **tag: callbacks (tag selection):**
    - `tag:cancel` → same as nav:cancel
    - `tag:none` (per TAGS-02) → set `s.selectedTags = []`, set `s.state = 'awaiting_title'`, show title keyboard with autoTitle suggestion
    - `tag:done` → set `s.state = 'awaiting_title'`, show title keyboard
    - `tag:custom` (per TAGS-03) → set `s.state = 'awaiting_custom_tags'`, reply "Напишите теги через запятую:"
    - `tag:{tagName}` → toggle tag in `s.selectedTags` (add if not present, remove if present), re-render tag keyboard via editMessageReplyMarkup

    **title: callbacks:**
    - `title:skip` → `doSave(ctx, autoTitle(s.text || contentType + ' от ' + new Date().toLocaleDateString('ru')))`

    **New `doSave(ctx, title)` function** replacing `saveToNotion()`:
    1. Reply "Сохраняю..."
    2. Build filename via `buildFilename(title, date)`
    3. Check duplicate via `findDuplicate(s.selectedFolder, filename)` (per DEDU-01)
    4. If duplicate: show duplicate keyboard (per DEDU-02), return (don't save yet)
    5. Prepare photo buffers from `pendingPhotos.get(ctx.chat!.id)`:
       - For each buffer: generate name `img_${Date.now()}_${contentHash(buffer.toString('base64')).slice(0, 8)}.jpg`
       - Build `{ name, buffer }[]` array
    6. Call `saveEntry(data, photoBuffers)` with all session data
    7. Reply with confirmation: "Сохранено: {title}\n{folder}{filename}\nТеги: {tags or 'нет'}"
    8. Clean up: `pendingPhotos.delete(ctx.chat!.id)`, reset session

    **Error handling:**
    - Wrap doSave in try/catch, report errors to user

    - Keep owner guard unchanged
    - Keep /start and /cancel commands, update /start text to mention "Obsidian vault" instead of "Notion"
